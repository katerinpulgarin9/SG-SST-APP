# SG-SST — Gestión documental (Streamlit)

## Ejecutar

Desde esta carpeta `sst-app/`:

```bash
pip install -r requirements.txt
streamlit run app.py
```

O desde la carpeta padre: doble clic en `INICIAR_APP.bat`.

La app abre en http://localhost:8501

## Notas

- Base de datos local: `data/empresa.db`
- Control maestro: `data/FG-FOR-SST-01_CONTROL_MAESTRO_DOCUMENTAL.xlsx`
- Plantillas del usuario: `plantillas/usuario/`
- Pack base: carpeta `PLANTILLAS/` en el directorio padre del proyecto
